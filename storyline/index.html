# 正文

--太初时代，在比传说更遥远的时间中，万物的框架刚刚搭建完成时，三个被神选中的生命被赋予了神的一部分力量，作为神的代言人成为了监督者。三名监督者依据自己对生命前进的看法创造了各自的世界，一个被称作“枫叶（Maple）”，一个被称作“格兰帝斯（Grandis）”，还有一个至今未被发现，三个世界以各自的方向开始了自己的历程。

--三个世界的物种由它们的监督者创造并不断完善。随着世界的逐步成熟，监督者们认为自己已经没有必要再直接影响世界的进程，为了确保世界的正确运行方向并能对其保持观察，监督者们向各自的世界都配备了三位超越者，使其作为自己的代理者，以作为在凡人眼中近乎于“神”的地位管理各自的领域：时间、生命、光明。超越者们被赋予管理世界的使命，但当超越者出于某种原因堕落后，则会转化为毁灭者，成为毁灭者后其对应的领域也会逐渐崩溃。同时，出于未知原因，监督者在世界中留下了诞生第四位超越者的可能，之后，三名监督者便消失在了传说之前的历史中。

--在Maple世界中，三位超越者分别为伦娜，管理时间；阿丽莎，管理生命；以及某个没有被提及名字的人，掌管光明；“奥西利亚大陆”从此建成并成为连接整个冒险岛世界的超级大陆。 在Grandis世界中，艾欧娜、克洛尼卡成为光明和时间超越者，首任生命超越者暂时不明。

引用诺巴族大神官的话“超越者算是‘半神’，有的超越者是不灭的，而有的超越者生来就注定面临毁灭”。

太初时代已知剧情至此结束。

--监督者创世完成后，生命超越者“阿丽莎”以实际体验的方式了解世界中生命的运行，居住于以自己力量创造的“世界树”内并以此被世人所尊称。

时间超越者“伦娜”在亲身与世界接触一段时间，并被尊奉为“时间女神”之后，为了以更客观的姿态管理时间的进程，使自己的肉体陷入沉睡，而以精神体的形式观测世间的事物与时间流的正确性，肉体则进入长久的沉睡；

同时，“神兽”作为被监督者赋予协助管理世界万物发展的圣物和诞生第四位超越者的线索，与其同时被创造的还有一支能运用并引导神兽力量的血脉。出于其崇高地位，该一族的领导者被冠以“皇帝”这一最高称号，并居住于由神兽力量庇护的圣地漂浮于大陆上空，其直属的追随者则以“圣地骑士团”所称。

（特别注释：事实上，即便对于是被称作“未曾睁开过双眼，我们的世界只是她的一场梦”的伦娜，在神之子任务中也提到了存在“比伦娜女神更古老的神灵”，各个现任超越者究竟是何时诞生的还不能直接确定，故本段内容请读者自行取舍。）

  • 时间神殿在由时间超越者“伦娜”所选中的一批追随者修建完成后，为了保护伦娜肉体的安全漂浮于世界上空。

  • 大陆开始长时间和平时代。

~ Grandis世界中，光明超越者“艾欧娜”自创世后便此久避于世，从此失去踪迹（传说中居住在只有光能到达的光之圣殿内）。时间超越者“克洛尼卡”则居于为自己所建的“纪年圣殿”中。

  • 两千年以上的自然发展时期。 （另注：在幻影剧情中，利琳提到“记录往往可以流传上千年，甚至更长的时间”，也就是说在上古之战前已经有了千年乃至更久的历史记录，故Maple世界应当已经有了两千年之上的历史，同时3个世界的创世时间应相差无几，故Grandis世界同理。）

  • 光明超越者进行下一任的交替（转至即我们所熟知的“白魔法师”）。 （另注：因为其以“白魔法师”的身份存在只有几十年的记录，且是通过“天赋与学习”达到了能被称作“白魔法师”的高度而非长久隐居突然出现，故本文于此认为白魔法师并非首任光明超越者。）

  • 在对时空魔法进行足够的开发研究后，魔法师们成功通过魔法将部分自传型书籍的内容具现化，使得阅读者可以亲身体验记录着的经历，并在某个空间内创造了“次元图书馆”以收集并保存诸如此类的有特殊意义的书籍，同时，该图书馆也被用于记录“历史性”的事件，该空间与魔法密林的“魔法密林图书馆”相连接。 （另注：次元图书馆更精确的创立时间待定。）

  • “欧罗拉”组织在白魔法师带领下在现艾琳森林的位置附近修建了秘密聚集地“极光神殿”，并在此进行“极光”的研究。

  • 双弩精灵“Mercedes（梅赛德斯）”的父母离世。

~ 在Grandis世界的种族竞争中，种族天生优势较强的高等精灵族和树精族为使自己种族拥有更高地位而交战，两族从此结下世仇并战火不断，在漫长的战争中，双方逐渐有了蔑视其他生命的看法。

  • 作为Maple世界巅峰的光之魔法师，光明超越者以“白魔法师”被世人尊称。但由于长时间的试验始终无法有突破性进展，光明超越者在留下“极光只能在极暗中才能发现”的话后隐居于极光神殿内，并开始尝试使用禁忌寻找极光的存在，由于该试验的巨大负面影响，黑暗生物“乌曼”在世界各地出现。

  • 在试验的过程中，白魔法师的善良和对于理想的热情也获得了居住于现艾琳森林的蝴蝶女皇“艾菲尼亚”的好感。

（另注：“奥尔卡&斯乌”的原身份黑暗精灵中的“精灵”对应的英文为Elemental，而双弩精灵“Mercedes”中的“精灵”为Elf，前者为元素精灵，后者为传统意义上的精灵，而“古瓦洛”和“Mercedes”分别为前者和后者相对应的精灵王。）

  • “欧罗拉”组织随后明白了这个结晶的意义，为了阻止其自然消散，欧罗拉组织使用自己领悟到的光之力量将其保留使其能以光的形式留在世间，并将在久远的未来以生命形式重新诞生，欧罗拉组织将其称为“星之子”。

  • “欧罗拉”组织的首席魔法师“马尔斯”接任为“欧罗拉”组织的第二代宗师。

  • 为了防止力量恢复后的黑魔法师伤害将诞生的星之子，“欧罗拉”组织将据点迁移到与现实世界分离的独立次元空间“希利尼提”内，同时为了防止组织寻找到星之子时黑魔法师的偷袭，组织选中了女孩“露西娅”，以其作为实力潜质最强大且与光之力量最亲和的人进行培养，准备在星之子诞生后将“露西娅”伪装为星之子被黑魔法师杀死以打消其顾虑，同时，作为给予“露西娅”注定死亡的命运的补偿，“欧罗拉”组织用所领悟的知识给予其第二次转生的能力。

# 幻影的管家的雇佣时间？

问：幻影的管家的雇佣时间？

答：正反两边的证据如下。 管家在500年前就雇佣了：管家知道渡鸦用药过度而死的事实，管家口述“当初以为少爷只是一个嚷着要冒险的富家公子”。 管家在幻影苏醒后才聘用：普通人基本不可能活过500年，管家口述“没想到少爷居然是过去的英雄”，幻影的宝库百年来并没有人照顾，有了水晶花园（剧情中已确认是苏醒后购买）才有了管家的价值。 经剧情小组讨论后，结论倾向于“管家在幻影苏醒后才聘用”这个观点。

  1. 渡鸦作为非常知名的怪盗，他的行动和突然失踪的缘由当时应该有很多人关心，也就有了被书本记载的可能（这个时期的记录斯乌没有去删除）。那么作为一个目前表现的非常的尽职的管家，因为主人是渡鸦的弟子而特意去查找渡鸦的记录，这一点可以说通。
  2. 管家并没有类似龙神的契约绑定和双弩的种族连带这种特性，如果说诅咒会因接触过近而扩散，那么十字军团任务中的成员在于战神等英雄的接触中应当也守影响，但事实相反。而即便是炼金和魔法造诣非凡的冰封雪域的“阿尔卡斯特”，也只是经历了三百年便白发苍苍，同为人类的管家理论上是活不到500年的。
  3. “管家口述‘当初以为少爷只是一个嚷着要冒险的富家公子’”这一点，因为幻影与管家见面时由于诅咒影响实力仅为10级，又有大量金钱挥霍，是有可能产生这样的理解的，这个“当初”也存在翻译模糊，无法辨认出是否指“上古时期”。 综上，各证据相对更倾向于后者观点，故得出如上结论。

# “品克缤（PB）”难道不是女神“伦娜”的宠物吗？

问：“品克缤（PB）”难道不是女神“伦娜”的宠物吗？

答：PB是女神的宠物的说法取自于它刚刚成为boss时的称号“时间的宠儿——品克缤”，但未指明这个“时间”是时间超越者“伦娜”。在神之子剧情中Alpha& Beta曾与PB遭遇，且感觉到这是“像来自其余世界的与时间之力截然不同的存在”，并向神官们说明时，神官称之为“比女神（即伦娜）更古老的过去神灵的残渣”。这个“更古老的过去神灵”是指伦娜之前的时间超越者，或是其他世界来的时间超越者，甚至是在那之上的时间监督者都未能可知。唯一能确定的是，这位“时间的宠儿”并非当下的时间超越者“伦娜”的宠物。

# 冒险岛ds剧情为什么不纳入正史呢？

问：冒险岛ds剧情为什么不纳入正史呢？

答：ds剧情中确实出现了很多精彩到让我们难以割舍的内容，最著名的便是其中的闹钟（帕拉图斯）剧情。但与正史的对比中，我们遗憾的发现其中 “四源罪”“冒险岛创世女神”等设定过于超纲（正史中无相关描述），且这种在ds设定中三千年前就存在（远早于上古之战）的设定一旦写入正史会产生先入的影响，无法满足“作为不过多涉入主线的分支剧情”的条件。因为N社编写ds剧情的时间较早，无法跟随当下剧情的发展，虽然N社在日后可能将其修改后放入正史（像旧版的逆奥之城一样），但在眼下，不要在研究正史时过多考虑ds剧情或许是更明智的选择。

# 所谓“Red冒险家”“V111版骑士团”“新版暗影双刀”是如何定义的呢？

问：所谓“Red冒险家”“V111版骑士团”“新版暗影双刀”是如何定义的呢？

答：类似于在第一问中对冒险家关系的解释，“新版骑士团”跟“Red冒险家”一样，是顺承旧版骑士团剧情进行的，这一点在其出生的圣地剧情中就能有所体会（旧版骑士团亲身经历了变形术士和黑魔女对圣地的入侵，而新版骑士则以模拟训练等方式“感受”），即骑士团也有第一代与第二代之分。但与冒险家和骑士团不同，新版暗影双刀职业则是将原剧情进行了推翻和修改，即进行了重编后覆盖掉了旧版暗影双刀，并无第一代与第二代之分。

# 夜光能和神之子一样视作未觉醒的超越者吗？

问：夜光能和神之子一样视作未觉醒的超越者吗？

答：夜光是从“黑魔法师由白魔法师转化后残存的纯粹的光结晶”中诞生的，这一点与神之子从“伦娜用力量凝结成的眼泪”中诞生有一定相似性。然而，毕竟伦娜的出发点是为了创造继任者，黑魔法师则是单纯将无法转化的黑暗的部分剔除，再加上转化为夜光的光结晶只是相对光明超越者全体力量中“残存”的极小一部分了，故与神之子相比还是有很明显的差距的。或许若黑魔法师死亡导致光明超越者缺失时，夜光有晋升为超越者的可能，但在目前来看他的身世仅仅是能赋予他使用光魔法的体质，还无法算入神之子那种准超越者级别。

# 许多剧情副本里不同职业都有独立剧情，那么这时候在正史里应视作谁参加的呢？

问：许多剧情副本里不同职业都有独立剧情，那么这时候在正史里应视作谁参加的呢？

答：对于大多数剧情副本，参与的人数都不止一人（如阿卡伊勒的是整个十字军团，黑色天堂的是联盟的各个组织），这时候我们可以视作他们都参加了，但玩家在不同阶段扮演的角色并不唯一，或是多数职业同时参与，例如，在黑色天堂中，我们在做独立剧情时扮演的是对应的职业，但之后剧情里以“队长”身份代理圣地的三只小鸡时则是所有账号统一扮演另一人，换句话说，这个剧情中“队长”仅有一人，是某个不确定的联盟成员，而其余职业并非这个“队长”，只是单纯的参战成员，其它剧情副本同理（例如数名十字军成员一起在阿卡伊勒副本里帮助受伤的英雄们）；唯一例外的是，在艾琳森林副本里因为“卡奥”仅是一人，所以回到过去的应也只有一人（应道是某个冒险家），虽然战神和夜光都有独立剧情，但我们此时应将其视为另一段剧情（即并非扮演“卡奥”，而是带着各自的目的穿越到过去探寻信息）。

# 冒险家“卡奥”到底是谁？

问：冒险家“卡奥”到底是谁？

答：我们先列出两人：原来的卡奥，Red冒险家A。 首先我们要知道，混血儿“提斯（凯琳的哥哥）”在上古时期就出生了，然后如下。 原来的卡奥：在上古之战时期出生，结识混血儿“提斯”，卷入时间漩涡，失忆，被“时间神殿管理员”搭救，成为时间神官。 Red冒险家A（长相与卡奥相同）：在现世出生，被混血儿“提斯”当作卡奥，成长（冒险之书任务），进行艾琳森林任务（也代替原来的卡奥与提斯互动），回到现世，进行时间神殿任务（发现本来的卡奥），完成冒险之书任务。 也就是说，提斯在上古时期就认识了原本的卡奥，在卡奥消失在时间漩涡后又结识了从现世穿越过去的Red冒险家A；500年过后，提斯在彩虹岛遇到了刚刚来到Maple世界的Red冒险家A（但是此时该冒险家A初来冒险岛，还没有进行艾琳森林任务，自然也就不认识提斯）。 而对于原来的卡奥，他只是与提斯有过一段时间交集，之后就卷入时间漩涡失忆当神官了，再也没有直接参与其余剧情中（当然也不排除参与日后剧情的可能）。 最后至于艾琳森林的战神和夜光的独立剧情，参考问答9，仅为回到过去寻找某些东西，从头至尾也没参与有关卡奥的活动。
